# Contributing to ShadowChat v1111

Welcome to the Sovereign State of ShadowChat. We welcome contributions to our digital infrastructure.

## 🚀 Getting Started

1. **Fork the Repository**: Create your own copy of the codebase.
2. **Clone the Repo**: `git clone https://github.com/skylerblue4444/shadowchat-v111-enterprise.git`
3. **Install Dependencies**: `pnpm install`
4. **Run Dev Server**: `./start-dev.sh`

## 🛠️ Contribution Guidelines

- **Sovereign Code Style**: Use TypeScript and follow the glassmorphic, dark-luxury CIS standards.
- **Neural Core Sync**: All global state changes MUST be integrated via `useNeuralCore`.
- **Unit Testing**: Ensure all new modules pass the `pnpm build` check.
- **Pull Requests**: Provide detailed descriptions of your infrastructure upgrades.

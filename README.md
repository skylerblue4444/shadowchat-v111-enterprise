## 🚀 Quick Start

### Development Mode (Recommended for Exploration)
```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Access at http://localhost:5173/
```

### Production Build
```bash
# Install dependencies
pnpm install

# Seed the ecosystem
pnpm seed  # Populates the "Alive" data engine

# Build for production
pnpm build

# Launch the OS
pnpm start
```
